# Frontend (Chrome Extension App)

The frontend is the Chrome extension UI that users install and run on `claude.ai`.

Distribution model:
- Package extension build and share install/download link
- Users install extension in Chrome and it activates on Claude.ai
- Sidebar UI is bundled with extension (recommended React + TypeScript)

Scope:
- prompt preview panel
- clarification Q&A panel
- improved prompt output
- skill recommendations view
- generated file preview/download actions
