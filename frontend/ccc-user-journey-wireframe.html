Claude Capability Coach - User Journey Wireframe

Low-fidelity flowchart showing what the user sees, inputs, and gets as outputs from start to finish.

Screen Node Input Point Decision Branch Output/Deliverable

1) Claude.ai Page Loaded

User opens Claude web app. Coach button is floating on right side; sidebar hidden by default.

System state: Idle
→

2) Toggle Coach

User clicks Coach button. Sidebar slides in.

Action: Open/Close sidebar
→

3) Home Screen

Shows title + guidance + Start button.

Input: Click Start
→

4) Chat Input

User types task prompt manually (no auto-read from Claude textbox).

Example inputs: "Indian capital", "analyse and prepare a PPT..."
→

5) Intent Detection

Coach checks if task is simple or complex.

Decision: Simple vs Complex route

BRANCH A - SIMPLE TASK PATH

A1) Simple Result

Coach explains quick task. User can end session or generate improved prompt anyway.

Input: Generate Prompt Anyway / End
→

A2) Prompt Output

Structured prompt displayed.

Output: Better prompt text
→

A3) Copy + Use in Claude

User copies prompt and pastes into Claude chat.

End output: Immediate execution in Claude.ai

BRANCH B - COMPLEX TASK PATH

B1) Clarifying Questions

Step-by-step multiple-choice questions (frequency, format, detail level).

Input: Option selections
→

B2) Intent Editor

User edits goal, instructions, output format, tone, and notes.

Input: Editable fields
→

B3) Prompt Output

Coach generates high-quality structured prompt from intent model.

Output: Better prompt + copy action
→

B4) Skill Decision

Ask: "Create reusable skill package?"

Decision: Yes / No
→

B5) Skill Generator

Generates package with `SKILL.md`, `instructions.md`, references, upload guide.

Output: Downloadable ZIP
→

B6) Reuse + Upload

User downloads ZIP and uploads as reusable skill in Customize/Skills workflow.

End output: Reusable workflow asset

Key Journey Notes

Primary User Inputs

  • Toggle Coach button click
  • Task prompt text
  • Question option selections
  • Intent editor field updates
  • Skill creation decision

Primary Outputs

  • Complexity classification
  • Refined intent model
  • Generated structured prompt
  • Suggested reusable skills
  • Skill package ZIP files

Navigation Behavior

  • Back button available on all screens
  • New Session resets complete flow
  • Toggle can hide/show sidebar anytime

Purpose of This Wireframe

  • Understand sequence quickly
  • Review where data is collected
  • Review where deliverables are produced
  • Use as blueprint before visual polish
